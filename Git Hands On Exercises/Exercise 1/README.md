# Git Hands-On Exercise 1 - Git Repository Initialization

## Overview

This exercise introduces the basic Git workflow by configuring Git, initializing a local repository, creating files, staging changes, committing them, and synchronizing with a remote GitHub repository.

## Objectives

- Configure Git username and email.
- Initialize a Git repository.
- Track files using Git.
- Commit changes.
- Push and pull changes from GitHub.

## Commands Covered

- git --version
- git config
- git init
- git status
- git add
- git commit
- git push
- git pull

## Technologies Used

- Git
- GitHub
- Git Bash
- Visual Studio Code

## Procedure

1. Install Git.
2. Configure Git.
3. Initialize repository.
4. Create files.
5. Stage files.
6. Commit changes.
7. Connect GitHub repository.
8. Push and Pull repository.

## Learning Outcomes

After completing this exercise, I learned to:

- Configure Git.
- Create repositories.
- Stage and commit files.
- Work with remote repositories.
- Synchronize repositories using GitHub.
