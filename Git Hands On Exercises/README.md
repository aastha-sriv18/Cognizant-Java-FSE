# Git Hands-On Exercises

This directory contains the Git Hands-On Exercises completed as part of the **Java Full Stack Engineering (FSE)** learning program. These exercises demonstrate the fundamentals of Git, including repository management, version control, branching, merging, conflict resolution, and remote repository synchronization.

---

## Exercises Included

| Exercise | Title | Concepts Covered |
|----------|-------|------------------|
| Exercise 1 | Git Configuration and Repository Setup | Git installation, configuration, editor setup, repository initialization, staging, commit, push and pull |
| Exercise 2 | Using `.gitignore` | Ignoring unwanted files and folders, working directory management |
| Exercise 3 | Branching and Merging | Creating branches, switching branches, merging, viewing commit history |
| Exercise 4 | Merge Conflict Resolution | Conflict detection, resolving merge conflicts, updating `.gitignore`, branch cleanup |
| Exercise 5 | Clean Up and Push to Remote Repository | Repository cleanup, pulling latest changes, pushing commits to remote repository |

## Learning Outcomes

After completing these exercises, I learned to:

- Configure Git on a local machine.
- Initialize and manage Git repositories.
- Stage and commit project files.
- Ignore unnecessary files using `.gitignore`.
- Create and manage branches.
- Merge branches successfully.
- Resolve merge conflicts.
- Synchronize local repositories with remote repositories using GitHub.
- Maintain a clean and organized version history.
