# Git Hands-On Exercise 5 - Clean Up and Push to Remote Repository

## Overview

This exercise demonstrates how to synchronize a local Git repository with a remote repository. It includes verifying the repository status, listing branches, pulling the latest changes, and pushing local commits to the remote repository.

## Objectives

- Verify repository status.
- List available branches.
- Pull the latest changes from the remote repository.
- Push local commits to the remote repository.
- Confirm successful synchronization.

## Concepts Covered

- Remote Repository
- Git Pull
- Git Push
- Git Status
- Git Branch
