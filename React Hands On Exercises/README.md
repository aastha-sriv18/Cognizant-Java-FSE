# React Hands-On Exercises

This directory contains the complete source code for all React laboratory exercises completed as part of the **Cognizant Java Full Stack Engineer (FSE) Program**. Each exercise focuses on a specific React concept and is organized in its own project folder with an individual README explaining the implementation.

## Running an Exercise

Navigate to the required exercise folder.

```bash
cd ExerciseX/<project-name>
```

Install dependencies.

```bash
npm install
```

Run the application.

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```
