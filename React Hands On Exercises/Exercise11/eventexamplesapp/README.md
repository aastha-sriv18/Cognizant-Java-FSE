# Exercise 11 - Event Examples Application

## Overview

This exercise demonstrates event handling in React. Various button click events and a currency converter are implemented to understand React's synthetic event system.

## Objectives

- Handle events in React.
- Use event handlers.
- Implement synthetic events.
- Handle user input.

## Features

- Increment Counter
- Decrement Counter
- Welcome Button
- Synthetic Event Button
- Currency Converter (Euro to INR)

## Technologies Used

- React.js
- JavaScript (ES6)
- JSX

## Expected Output

The application allows users to:

- Increment and decrement a counter.
- Display greeting messages.
- Trigger synthetic events.
- Convert Euro values into Indian Rupees.

- Implement click events.
- Pass parameters to event handlers.
