# Exercise 10 - Office Space Rental Application

## Overview

This exercise demonstrates JSX syntax and rendering dynamic content in React. The application displays office spaces with rent, address, and images while applying conditional styling.

## Objectives

- Learn JSX syntax.
- Render objects and arrays.
- Apply inline CSS.
- Use JavaScript expressions inside JSX.

## Features

- Displays office space information.
- Shows office images.
- Conditional rent coloring.
- Dynamic rendering using map().

## Technologies Used

- React.js
- JSX
- JavaScript (ES6)
- CSS

## Expected Output

Displays:

- Office Name
- Address
- Rent
- Office Image

Rent appears:

- **Red** if below ₹60,000
- **Green** if above ₹60,000
