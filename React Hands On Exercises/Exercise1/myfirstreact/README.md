# Exercise 1 - My First React Application

## Overview

This exercise demonstrates the basic setup of a React application using **Create React App**. It introduces the React development environment and explains how a simple functional component can be rendered to the browser.

The application displays a heading:

> **Welcome to the First Session of React**

## Objectives

- Set up a React development environment.
- Create a React application using Create React App.
- Understand the basic React project structure.
- Create and render a simple functional component.
- Display JSX content in the browser.

## Features

- React application created using Create React App.
- Functional component implementation.
- JSX rendering.
- Basic React project structure.

## Technologies Used

- React.js
- JavaScript (ES6)
- JSX
- HTML5
- CSS3
- Node.js
- npm

## Installation

Install dependencies.

```bash
npm install
```

## Running the Project

Start the development server.

```bash
npm start
```

Open your browser and navigate to:

```
http://localhost:3000
```

## Expected Output

The application displays the following heading:

```
Welcome to the First Session of React
```
