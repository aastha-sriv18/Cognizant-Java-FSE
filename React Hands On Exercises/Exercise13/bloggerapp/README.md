# Exercise 13 - Blogger Application

## Overview

This exercise demonstrates conditional rendering, list rendering, and the use of unique keys in React. The application displays Book Details, Blog Details, and Course Details using reusable components.

## Objectives

- Implement multiple conditional rendering techniques.
- Render lists using map().
- Use unique keys.
- Create reusable components.

## Features

- Book Details
- Blog Details
- Course Details
- Conditional Rendering
- List Rendering using map()
- React Keys

## Technologies Used

- React.js
- JavaScript (ES6)
- JSX
