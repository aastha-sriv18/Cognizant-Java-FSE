# Exercise 2 - Student Management Portal

## Overview

This exercise demonstrates the creation and rendering of multiple React components. A simple Student Management Portal is developed using reusable functional components to display the Home, About, and Contact pages.

The application introduces component-based architecture, one of the core concepts of React.

## Objectives

- Understand React components.
- Create multiple reusable functional components.
- Render components within a parent component.
- Learn component composition in React.

## Features

- Home component
- About component
- Contact component
- Component rendering using App.js
- Simple and reusable component structure

## Technologies Used

- React.js
- JavaScript (ES6)
- JSX
- HTML5
- CSS3
- Node.js
- npm

## Installation

Install all project dependencies.

```bash
npm install
```

## Running the Project

Start the React development server.

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
